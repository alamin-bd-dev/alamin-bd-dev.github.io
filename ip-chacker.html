About Ip Checker

This is a simple Python-based IP Information Checker that uses :contentReference[oaicite:0]{index=0} to fetch geolocation details from a public IP address.

Users can input an IP address and get information like:

🌎 Country

🏙 City

🏢 ISP

📍 Latitude & Longitude

How to run

$pip install requests

#Clone the project:

$git clone https://github.com/alamin-bd-dev/ip-checker

$cd ip-checker

$python Ip cheaker.py
